Voxbi

Connect this Odoo to Mixvoip Voxbi telephony in one step

What this module does

Voxbi adds a Voxbi → Configuration wizard to your Odoo backend. You paste a one-time install token generated in Mixvoip Cockpit and click Install. The wizard registers this Odoo with Mixvoip Cockpit, which then provisions and configures the Voxbi telephony integration for your database. Installation progress streams back into the wizard in real time.

Key features

One-time token setup

No credentials to type. Generate a single-use token in Mixvoip Cockpit, paste it, and install. Each token works once and can be revoked at any time.

Live progress

The wizard polls the install status and shows a color-coded log console, so you can watch the provisioning happen step by step.

Optional SIP sync

Opt in to push SIP settings into Odoo's VoIP module after install, or keep your existing VoIP configuration untouched.

Admin-only

The setup wizard is restricted to Settings administrators. Connection identifiers are masked in the UI by default.

How it works

  1. In Mixvoip Cockpit, open the user's settings page and go to the Odoo installer tab, then generate a single-use token under Odoo install tokens.
  2. In Odoo, open Voxbi → Configuration, paste the token, review the data-sharing notice, tick the authorization box, and click Install.
  3. The wizard shows live progress and confirms once Mixvoip Cockpit has configured the integration.

External service & data sharing

This module requires Mixvoip Cockpit, an external service operated by Mixvoip SA, to provision the Voxbi integration. When you click Install, the module asks for your explicit authorization and then sends the following data to Mixvoip Cockpit over an encrypted (HTTPS) connection:

  • The one-time install token you pasted
  • This Odoo instance's URL, database name and version
  • The login of the administrator running the wizard
  • A freshly generated Odoo API key for that administrator, used by Mixvoip Cockpit to configure the integration over XML-RPC
  • Your chosen options (SIP sync, active flag)

The generated API key can be revoked at any time from the administrator's Preferences → Account Security. You retain full ownership of your data. A Mixvoip account and an active Voxbi subscription are required to use this integration.
